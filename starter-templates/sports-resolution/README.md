# Sports Resolution

Resolve sports game outcomes on-chain using CRE: EVM log trigger → HTTP fetches to 2–3 sports APIs → majority/unanimous aggregation → signed report written back on-chain. Domain-agnostic pattern reusable for any binary or categorical resolution use case.
